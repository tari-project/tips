# Contributing to Tari Improvement Proposals (TIPs)

Thank you for your interest in contributing to the Tari blockchain! This document provides guidelines for contributing TIPs.

## How to Contribute a TIP

### 1. Discuss Your Idea

Before writing a formal TIP, discuss your idea with the Tari community:
- Join discussions on the [Tari Discord](https://discord.gg/tari)
- Open a GitHub issue to gauge interest

### 2. Write Your TIP

Once you have community interest:

1. **Fork this repository**
2. **Copy the TIP template**: `cp TIP-template.md tips/TIP-XXXX.md`
   - Use the next available number for XXXX (check the README for the latest)
3. **Fill in all sections** of the template thoroughly
4. **Follow the format** shown in the example TIPs

### 3. Submit Your TIP

1. **Create a pull request** with your TIP
2. **Update the README.md** to include your TIP in the table of contents
3. **Set the status** to "Draft"
4. **Be responsive** to feedback and questions

### 4. Review Process

- The community will review your TIP
- Core developers will provide technical feedback
- You may need to revise your TIP based on feedback
- Discussion will happen in the pull request

### 5. TIP Acceptance

Once your TIP is accepted:
- It will be merged into the repository
- Status will be updated to "Accepted"
- Implementation can begin
- When deployed, status becomes "Active"

## TIP Quality Standards

Your TIP should be:

- **Clear**: Easy to understand for developers
- **Detailed**: Sufficient technical specification
- **Well-motivated**: Clear explanation of the problem
- **Complete**: All template sections filled out
- **Correct**: Technically sound and feasible

## Writing Style

- Use clear, concise language
- Avoid jargon when possible
- Include code examples where helpful
- Use proper markdown formatting
- Check spelling and grammar

## Code of Conduct

All contributors are expected to:
- Be respectful and constructive
- Focus on technical merit
- Accept feedback gracefully
- Help others in the community

## Questions?

If you have questions about the TIP process:
- Read TIP-0001 (TIP Process and Guidelines)
- Ask in the community channels
- Open a GitHub issue with the "question" label
